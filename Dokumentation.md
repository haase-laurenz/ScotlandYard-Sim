# GameManager-Klasse

Die `GameManager`-Klasse ist verantwortlich für die Verwaltung und Ausführung von Spielen im Kontext des Detektiv-MisterX-Spiels.

## Konstruktor

```java
public GameManager(){
    gameGUI = new GameGUI(this);
}
```
